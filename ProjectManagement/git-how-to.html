Manage github projects

  1. git clone [remote-repo-name]
  2. git checkout -b [branchName]
  3. File structure should be downloaded. Edit, test, etc... When done...
  4. git add .
  5. git commit -a -m "Type comments between double quotes"
  6. git push -u origin [branchName]
  7. git checkout master
  8. git merge [branchName]
  9. git rebase
